t3tools.tucker_tensor_train.t3_scale¶

t3tools.tucker_tensor_train.t3_scale(x: TuckerTensorTrain, s) → TuckerTensorTrain¶

Scale TuckerTensorTrain x by a scaling factor s.

Parameters:

  • x (TuckerTensorTrain) – Tucker tensor train

  • s (scalar) – scaling factor

Returns:

z – scaled TuckerTensorTrain z=s*x, with the same structure as x.

Return type:

TuckerTensorTrain

Raises:

RuntimeError –

  • Error raised if the TuckerTensorTrains are internally inconsistent

See also

TuckerTensorTrain, t3_add, t3_neg, t3_sub

Notes

Scales the last basis core of x, leaving all other cores unchanged

Examples

>>> import numpy as np
>>> import t3tools.tucker_tensor_train as t3
>>> x = t3.t3_corewise_randn(((14,15,16), (4,5,6), (1,3,2,1)))
>>> s = 3.2
>>> z = t3.t3_scale(x, s)
>>> print(np.linalg.norm(s*t3.t3_to_dense(x) - t3.t3_to_dense(z)))
1.6268482531988893e-13
