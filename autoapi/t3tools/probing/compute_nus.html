t3tools.probing.compute_nus¶

t3tools.probing.compute_nus(right_tt_cores: Sequence[t3tools.common.NDArray], xis, use_jax: bool = False) → Sequence[t3tools.common.NDArray]¶

Compute rightward edge variables associated with edges between adjacent TT-cores. Used for probing a Tucker tensor train.

See Section 5.2, particularly Figure 9 in:

Alger, N., Christierson, B., Chen, P., & Ghattas, O. (2026). “Tucker Tensor Train Taylor Series.” arXiv preprint arXiv:2603.21141. https://arxiv.org/abs/2603.21141

Parameters:

  • right_tt_cores (typ.Sequence[NDArray]) – Right TT-cores for Tucker tensor train. len=d. elm_shape=(ri,ni,r(i+1))

  • xis (typ.Sequence[NDArray]) – upward edge variables xi. len=d. elm_shape=(num_probes,ni)

  • use_jax (bool) – If True, use jax for linear algebra operations. Otherwise, use numpy.

Returns:

rightward edge variables nu. len=d+1, elm_shape=(num_probes,ri)

Return type:

typ.Tuple[NDArray,…]

See also

probe_t3, compute_xis, compute_mus, compute_etas, assemble_probes
