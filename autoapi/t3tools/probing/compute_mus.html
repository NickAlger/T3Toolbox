t3tools.probing.compute_mus¶

t3tools.probing.compute_mus(left_tt_cores: Sequence[NDArray], xis: Sequence[NDArray], use_jax: bool = False) → Sequence[NDArray]¶

Compute leftward edge variables associated with edges between adjacent TT-cores. Used for probing a Tucker tensor train.

See Section 5.2, particularly Figure 9 in:

Alger, N., Christierson, B., Chen, P., & Ghattas, O. (2026). “Tucker Tensor Train Taylor Series.” arXiv preprint arXiv:2603.21141. https://arxiv.org/abs/2603.21141

Parameters:

  • left_tt_cores (typ.Sequence[NDArray]) – Left TT-cores for Tucker tensor train. len=d. elm_shape=(ri,ni,r(i+1))

  • xis (typ.Sequence[NDArray]) – upward edge variables xi. len=d. elm_shape=(num_probes,ni)

  • use_jax (bool) – If True, use jax for linear algebra operations. Otherwise, use numpy.

Returns:

leftward edge variables mu. len=d+1, elm_shape=(num_probes,ri)

Return type:

typ.Tuple[NDArray,…]

See also

probe_t3, compute_xis, compute_nus, compute_etas, assemble_probes
