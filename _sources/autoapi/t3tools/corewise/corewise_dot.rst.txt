t3tools.corewise.corewise_dot
=============================

.. py:function:: t3tools.corewise.corewise_dot(X: NDArrayTree, Y: NDArrayTree, use_jax: bool = False)

   Dot product of nested objects, X,Y -> X.Y.

   .. rubric:: Examples

   >>> import numpy as np
   >>> import t3tools.corewise as cw
   >>> X = (np.ones(3), (1, (), np.ones(2)))
   >>> Y = (2*np.ones(3), (3, (), -np.ones(2)))
   >>> print(cw.corewise_dot(X, Y))
   7.0
