t3tools.corewise.corewise_norm
==============================

.. py:function:: t3tools.corewise.corewise_norm(X, xnp=np)

   Norm of nested objects, X -> ||X||

   .. rubric:: Examples

   >>> import numpy as np
   >>> import t3tools.corewise as cw
   >>> X = (np.ones(3), (1, (), np.ones(2)))
   >>> print(cw.corewise_norm(X))
   2.449489742783178
   >>> print(np.sqrt(3 + 1 + 2))
   2.449489742783178
