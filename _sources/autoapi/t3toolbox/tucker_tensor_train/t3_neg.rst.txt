t3toolbox.tucker_tensor_train.t3_neg
====================================

.. py:function:: t3toolbox.tucker_tensor_train.t3_neg(x: TuckerTensorTrain) -> TuckerTensorTrain

   Scale a Tucker tensor train by -1.

   Negation is defined with respect to the dense N0 x ... x N(d-1) tensor that
   is *represented* by the Tucker tensor trains, even though this dense tensor
   is not formed during computations.

   For corewise negation, see :func:`t3toolbox.corewise.corewise_neg`

   :param x: Tucker tensor train
   :type x: TuckerTensorTrain

   :returns: Negated TuckerTensorTrain -x, with the same structure as x.
   :rtype: TuckerTensorTrain

   :raises ValueError: - Error raised if the TuckerTensorTrains is internally inconsistent

   .. seealso:: :py:obj:`TuckerTensorTrain`, :py:obj:`t3_add`, :py:obj:`t3_scale`, :py:obj:`t3_sub`, :func:`~t3toolbox.corewise.corewise_neg`

   .. rubric:: Examples

   >>> import numpy as np
   >>> import t3toolbox.tucker_tensor_train as t3
   >>> x = t3.t3_corewise_randn(((14,15,16), (4,5,6), (1,3,2,1)))
   >>> neg_x = t3.t3_neg(x)
   >>> print(np.linalg.norm(t3.t3_to_dense(x) + t3.t3_to_dense(neg_x)))
   0.0
